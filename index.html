Fingerprint

Fingerprint a users browser

Download this project as a .zip file Download this project as a tar.gz file

finger.js

Drop-in privacy invasion. Add this to the <head> of your HTML and you'll get an HTTP cookie with each request that identifies the browser whether the visitor is logged in or not.

Since it's generated the same way every time, and uses a bunch of data that is fairly specific to each user - fonts installed, version of Java, version of Flash, user agent string, etc, etc. - you can identify the user even if they've deleted all cookies.

It's pretty much impossible to use this for good, and I've just pulled it together to show how trivial it is to track people without their consent. Sadface.
